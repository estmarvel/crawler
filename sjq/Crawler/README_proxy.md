# IP代理池调用方法
服务器地址223.109.239.18已经添加入白名单  
IP代理池文件：proxy_pool.py  
文件中默认参数等价于：
```text
http://api.tianqiip.com/getip?secret=aliejy64ogh6c1kx&num=10&type=json&port=2&time=3&mr=1&sign=e69907c179da76044ed221bdb095e0f5
```
含义：以JSON格式获取10个3分钟短效HTTPS协议代理IP  
## 1. 直接调用

```python
from proxy_pool import ProxyPool, ProxyPoolEmptyError

pool = ProxyPool()

try:
    proxy = pool.get_proxy()
    print(proxy)
except ProxyPoolEmptyError as e:
    print("无可用代理：", e)
```

返回格式：

```python
{
    "http": "http://121.227.60.236:20252",
    "https": "http://121.227.60.236:20252"
}
```

这个结果可以直接传给 `requests`：

```python
import requests

resp = requests.get(
    "https://www.ygcgpt.com",
    proxies=proxy,
    timeout=15,
    verify=False,
)
```

------

## 2. 自定义提取参数

初始化时自定义：

```python
pool = ProxyPool(num=20, time=5, port=2)
```

表示：每次从天启提取 20 个、5 分钟、HTTPS 代理。

也可以在获取代理时临时指定：

```python
proxy = pool.get_proxy(num=20, time=5, port=2)
```

支持的关键参数：

| 参数     | 说明                            |
| -------- | ------------------------------- |
| `num`    | 提取数量，1~200                 |
| `time`   | 有效期，只支持 3、5、10、15     |
| `port`   | `1` HTTP，`2` HTTPS，`3` SOCKS5 |
| `yys`    | 运营商：电信、移动、联通        |
| `region` | 区域编号                        |
| `ts`     | 是否显示过期时间，1 显示        |
| `ys`     | 是否显示运营商，1 显示          |
| `cs`     | 是否显示位置，1 显示            |
| `mr`     | 是否去重，1 去重                |

示例：

```python
proxy = pool.get_proxy(
    num=10,
    time=3,
    port=2,
    mr=1,
    ts=1,
)
```

------

### 标记坏代理

爬虫请求失败时调用：

```python
pool.mark_bad(proxy)
```

同一个代理连续失败达到阈值后，会自动从代理池移除。

------

### 查看代理池状态

```python
print(pool.stats())
```

返回示例：

```json
{
  "available": 8,
  "bad_total": 2,
  "api_calls_used": 1,
  "api_calls_limit": 200,
  "min_size": 3,
  "default_num": 10,
  "default_time": 3,
  "default_port": 2

```

------

### 手动刷新代理池

```python
pool.refresh(num=10, time=3, port=2)
```

该操作会清空当前代理池，并重新从天启 API 提取一批代理。

# 天启 IP 接口使用说明

## 1. 提取接口说明

### 1.1 基本信息

| 属性    | 内容                              |
| ----- | ------------------------------- |
| URL   | `http://api.tianqiip.com/getip` |
| 请求方法  | `GET`                           |
| 编码方式  | `utf-8`                         |
| 成功状态码 | `1000`                          |
| 返回类型  | `txt` / `json`                  |

---

## 2. 提取接口参数说明

| 参数名      | 是否必填   | 说明                                             |
| -------- | ------ | ---------------------------------------------- |
| `secret` | 是      | 提取秘钥                                           |
| `sign`   | 是      | 用户签名，提取时必须添加                                   |
| `num`    | 否      | 提取 IP 数量，不能超过 200，默认值为 10                      |
| `yys`    | 否      | 提取 IP 运营商，可选：电信、移动、联通                          |
| `type`   | 否      | 返回类型，可选：`txt`、`json`                           |
| `lb`     | 否      | `txt` 格式返回时的分隔符                                |
| `region` | 否      | 区域编号，多个区域用英文逗号分隔                               |
| `port`   | 是      | IP 协议类型：`1` 表示 HTTP，`2` 表示 HTTPS，`3` 表示 SOCKS5 |
| `time`   | 按次提取必填 | IP 使用时长，仅支持 `3`、`5`、`10`、`15`                  |
| `ts`     | 否      | 是否显示 IP 过期时间，`1` 表示显示，默认不显示                    |
| `ys`     | 否      | 是否显示 IP 运营商，`1` 表示显示，默认不显示                     |
| `cs`     | 否      | 是否显示位置，`1` 表示显示，默认不显示                          |
| `mr`     | 否      | IP 去重，`1` 表示去重，默认开启去重                          |

---

## 3. 提取接口返回说明

### 3.1 成功返回

| 字段     | 说明                  |
| ------ | ------------------- |
| `code` | 状态码，`1000` 表示获取成功   |
| `data` | IP 数据列表，仅 JSON 格式存在 |

---

### 3.2 JSON 返回字段说明

当 `type=json` 时，返回字段如下：

| 字段       | 说明                  |
| -------- | ------------------- |
| `ip`     | IP 地址               |
| `port`   | 端口                  |
| `expire` | 有效时间，需开启 `ts=1` 才显示 |
| `city`   | 城市                  |
| `prov`   | 省份                  |
| `isp`    | 运营商，需开启 `ys=1` 才显示  |

---

## 4. 提取接口返回示例

### 4.1 TEXT 格式示例

```text
114.101.252.126.57577
114.101.252.126.57577
114.101.252.126.57577
```

> 说明：TEXT 格式中的 IP 和端口分隔符可通过 `lb` 参数或产品管理中的提取格式进行定制。

---

### 4.2 JSON 格式示例

```json
{
  "code": 1000,
  "data": [
    {
      "ip": "121.227.60.236",
      "port": "20252",
      "prov": "辽宁",
      "city": "鞍山"
    },
    {
      "ip": "121.227.60.236",
      "port": "20252",
      "prov": "辽宁",
      "city": "鞍山"
    }
  ]
}
```

## 5. 提取接口错误码

| 错误码    | 说明          |
| ------ | ----------- |
| `1001` | 请求格式不正确     |
| `1002` | 单次请求数量超出最大值 |
| `1003` | 请求 KEY 异常   |
| `1004` | 套餐已过期       |
| `1005` | 套餐提取数量上限    |
| `1006` | 暂无可用 IP     |
| `1007` | 提取地区超出服务范围  |
