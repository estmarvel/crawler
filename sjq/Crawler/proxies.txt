#
# 以 # 开头的行为注释行, 会被忽略
#
# 获取代理IP的推荐渠道(付费):
#   - 快代理:  https://www.kuaidaili.com
#   - 芝麻代理: http://www.zhimaruanjian.com
#   - 站大爷:  https://www.zdaye.com
#
# 使用说明:
#   1. 将本文件中的示例代理替换为你自己的代理IP
#   2. 确保代理IP对目标站点 (prec.sxzwfw.gov.cn) 可达
#   3. 建议至少配置 10-20 个代理IP
#   4. 运行: python main.py --proxy-file proxies.txt
